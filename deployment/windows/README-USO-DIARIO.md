# Uso diario en Windows

## Encender la herramienta

Haga doble clic en:

- `01-ENCENDER.bat`

## Apagar la herramienta

Haga doble clic en:

- `02-APAGAR.bat`

## Actualizar la herramienta

Haga doble clic en:

- `03-ACTUALIZAR.bat`

## Diagnóstico

Si algo no funciona, ejecute:

- `06-DIAGNOSTICO.bat`

## Menú único

Si prefiere una sola entrada, use:

- `MENU.bat`

## Observación importante

Este despliegue de Windows construye una imagen local del fork desde el propio repositorio.
