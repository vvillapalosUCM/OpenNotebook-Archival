# Alcance funcional archivístico

## Casos de uso iniciales

1. Clasificación documental asistida.
2. Extracción de metadatos básicos.
3. Resumen técnico de expedientes.
4. Apoyo a descripción archivística.
5. Detección preliminar de datos sensibles.
6. Generación de borradores administrativos revisables.

## Salidas prioritarias

- título propuesto;
- fechas detectadas;
- entidades nombradas;
- materias o funciones probables;
- resumen técnico;
- propuesta de serie o tipología;
- alertas sobre datos personales o límites de acceso.

## Límites

Las salidas son propuestas. No sustituyen valoración profesional, calificación archivística, dictamen jurídico ni decisión de acceso.
